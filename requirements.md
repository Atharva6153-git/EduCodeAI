## 1. Project Overview
Code Samjhao AI is a learning-first AI-powered code explanation and debugging assistant designed especially for beginners and students in Bharat.  
The system helps users understand code line-by-line in simple language (English/Hinglish) with a focus on teaching concepts rather than just providing answers.

---

## 2. Problem Statement
Many students struggle to learn programming due to:
- Overly technical documentation
- Complex error messages
- Lack of beginner-friendly explanations
- English-only learning resources

Existing AI tools focus on productivity for developers, not on teaching beginners how and why code works.

---

## 3. Objectives
- Help beginners understand code clearly and confidently
- Reduce fear and confusion around programming
- Provide explanations in simple, beginner-friendly language
- Support Bharat-focused learning needs

---

## 4. Target Users
- First-year engineering students
- Diploma and ITI students
- Coding beginners and self-learners
- Students from Tier-2 and Tier-3 cities

---

## 5. Functional Requirements

### 5.1 Code Input
- Users can paste code into a text editor
- Supported languages:
  - Python
  - JavaScript
  - Java

### 5.2 Explanation Level Selection
- Beginner mode (very simple explanation + analogies)
- Intermediate mode (technical explanation)
- Debug mode (focus on errors and fixes)

### 5.3 AI Code Explanation
- Line-by-line explanation of code
- Explanation of why each line is used
- Highlight important programming concepts

### 5.4 Debugging Assistance
- Identify common errors
- Explain why the error occurred
- Suggest correct and beginner-friendly fixes
- Share best practices

### 5.5 Output Display
- Clear, readable explanation format
- Scrollable output area
- Option to copy explanation text

---

## 6. Non-Functional Requirements
- Simple and clean user interface
- Fast response time
- Beginner-friendly language
- Low learning curve
- Works on low-end devices

---

## 7. Constraints
- Prototype-level implementation
- Limited AI API usage
- Internet connectivity required
- No user authentication in initial version

---

## 8. Assumptions
- Users have basic internet access
- Users paste syntactically valid code
- English/Hinglish explanation is sufficient for prototype

---

## 9. Future Enhancements
- Voice-based explanations
- Regional language support
- Mobile application
- WhatsApp-based AI tutor
- Learning progress tracking
